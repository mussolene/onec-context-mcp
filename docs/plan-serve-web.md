## 4. Технические заметки

- **categories** vs **tree**: `categories.build_tree` даёт дерево с заголовками из HTML; `tree.build_tree` — обход по файлам. Нужно выбирать categories при наличии `__categories__`.
- **Ссылки**: в Markdown/HTML справки ссылки вида `href="path/to/topic.html"`. Нужно переписывать на `/content/path/to/topic.html` или на вызов loadContent.
- **Поиск**: endpoint `/api/search?q=...` → indexer.search_hybrid (semantic + keyword RRF), результаты — список path/title/text. UI: debounce 350ms, блок «По индексу Qdrant» над деревом.
- **Контейнер**: `./data/unpacked` монтируется в `/data`; `HELP_SERVE_DATA_DIR=/data`. Команда: `serve` без аргументов.
